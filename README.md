# TgBot
Команды для чата:
/start - запуск бота
/quiz - запуск викторины

Вынес данные викторины и все вспомогательные функции в свои отдельные файлы, в главном файле Bot оставил непосредственно его команды
Объеденил right_answer и wrong_answer в одну функцию, а коллбеком стал передавать надпись с кнопки, чтобы в чат выводить и текст выбранного ответа
Статистику ответов сохраняю в db-файл
